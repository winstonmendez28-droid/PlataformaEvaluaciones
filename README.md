Plataforma de Evaluaciones

Descripción
Proyecto académico desarrollado en ASP.NET Web Forms.
Permite crear evaluaciones, registrar respuestas y generar resultados por usuario. Incluye navegación, uso de Master Page, controles ASP.NET, validaciones y manejo básico de sesión.
Objetivo
Este proyecto tiene como objetivo presentar la estructura inicial de la plataforma, mostrando la correcta creación del proyecto, la implementación de la plantilla general, navegación entre pantallas, uso de controles y validaciones. Esta primera entrega sienta las bases para las siguientes funcionalidades que se integrarán durante el semestre.
Estructura del proyecto
•	Site.Master → Plantilla general para todas las páginas
•	Default.aspx → Página principal de inicio
•	Pantallas adicionales → 7 módulos representativos (por ejemplo: registro de usuario, crear evaluación, resultados, control de tiempo)
•	Controles ASP.NET → TextBox, DropDownList, GridView/ListView, Buttons
•	Validaciones → Campos obligatorios y server-side
•	Sesión/Cookies → Control básico de acceso y preferencias del usuario
Cómo ejecutar el proyecto
1.	Clonar el repositorio:
git clone https://github.com/winstonmendez28-droid/PlataformaEvaluaciones.git
2.	Abrir la solución .sln en Visual Studio 2022
3.	Compilar y ejecutar (F5)
